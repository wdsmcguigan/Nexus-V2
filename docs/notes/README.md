# docs/notes/ — Informal docs

This folder holds informal, exploratory, or external material. **Not source of truth.** When something here graduates to a real decision, copy the relevant content into the canonical doc and update the file here to point at the new home (or delete it).

## Current contents

| File | What it is |
|---|---|
| `panels-brainstorm.md` | Quick brainstorm of nice-to-have panels and integrations (notes/todo, PDF viewer, calendar variants, teleconferencing, third-party integrations). Originally an extension-less file. |
| `self-hosted-email-comparison.md` | External strategic analysis comparing Nexus-V2 to 19 self-hosted email projects (Stalwart, Mox, Mailcow, etc.). Generated by Gemini, May 2026. Useful for product positioning; not a technical commitment. |

## What does NOT belong here

- Architecture decisions → `docs/architecture.md`
- Epic checklists → `docs/epic-<N>-checklist.md`
- API references → `docs/ipc-api-reference.md` / `docs/database-reference.md`
- Known gaps / TODOs → `docs/known-gaps.md`
- User-facing docs → `docs/user-guide.md`
- Developer recipes → `docs/developer-guide.md`
